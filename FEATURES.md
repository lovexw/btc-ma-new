# 功能特性详解

## 🎯 核心功能

### 1. 比特币价格走势图表

#### 主要特性
- **实时数据展示** - 读取CSV文件中的历史价格数据
- **渐变填充** - 价格曲线下方使用蓝色渐变填充，视觉效果优雅
- **平滑曲线** - 启用平滑算法，曲线更加流畅自然
- **阴影效果** - 曲线带有柔和阴影，增强立体感

#### 交互功能
- **缩放功能**
  - 鼠标滚轮缩放
  - 拖拽滑块调整时间范围
  - 双击重置视图
- **十字准星**
  - 鼠标悬停显示详细数据
  - 同时显示价格和各条均线值
  - 深色半透明背景，信息清晰可读

### 2. 多周期移动平均线

#### 支持的均线周期
| 周期 | 名称 | 用途说明 |
|------|------|----------|
| 50日 | MA50 | **短期趋势**<br>- 捕捉短期价格动向<br>- 适合短线交易参考<br>- 反应灵敏，波动较大 |
| 200日 | MA200 | **经典中期趋势**<br>- 最常用的技术指标之一<br>- 牛熊市分界线<br>- 重要支撑/压力位 |
| 350日 | MA350 | **中长期趋势**<br>- 介于中期和长期之间<br>- 平滑短期波动<br>- 趋势更加稳定 |
| 700日 | MA700 | **长期趋势**<br>- 约2年周期<br>- 识别长期牛熊市<br>- 重要战略参考位 |
| 1000日 | MA1000 | **超长期趋势**<br>- 约3年周期<br>- 极其稳定的趋势线<br>- 长期投资者关注 |
| 1400日 | MA1400 | **极长期趋势**<br>- 约4年周期<br>- 对应比特币减半周期<br>- 周期性分析重要指标 |

#### 均线控制
- **开关按钮** - 每条均线都可独立显示/隐藏
- **颜色区分** - 每条均线使用不同颜色，易于识别
- **图例显示** - 图表顶部显示当前激活的均线

### 3. 均线监控面板

#### 卡片信息
每个均线监控卡片包含以下信息：

1. **均线标识** (左上角)
   - 显示均线名称（MA50、MA200等）
   - 使用对应的主题色

2. **涨跌指标** (右上角)
   - 显示均线日变化率
   - 绿色↑表示上涨
   - 红色↓表示下跌

3. **当前数值** (中间大字)
   - 显示均线当前精确价格
   - 千分位格式化
   - 保留2位小数

4. **价格偏离度** (底部)
   - 显示当前价格与该均线的偏离百分比
   - 绿色表示价格在均线上方
   - 红色表示价格在均线下方

5. **位置提示** (最底部)
   - "价格在均线上方" - 表示多头
   - "价格在均线下方" - 表示空头

#### 视觉效果
- **渐变背景** - 每个卡片使用对应均线颜色的渐变背景
- **悬浮动画** - 鼠标悬停时卡片上升，阴影增强
- **响应式布局** - 自动适配不同屏幕尺寸
  - 大屏：6列
  - 中屏：3列
  - 小屏：2列
  - 手机：1列

### 4. 数据统计面板

位于页面底部，显示三个关键指标：

1. **当前价格** (蓝色)
   - 实时比特币价格
   - 千分位格式化显示

2. **数据点数** (绿色)
   - 显示历史数据的总天数
   - 了解数据覆盖范围

3. **最新日期** (紫色)
   - 显示最新数据的日期
   - 确认数据是否为最新

## 🎨 设计理念

### 色彩系统
- **主色调**：蓝色系（#1890ff）- 专业、稳重
- **背景色**：浅灰色（#f5f7fa）- 柔和、舒适
- **卡片色**：白色 - 清晰、简洁
- **渐变色**：紫色渐变标题 - 现代、醒目

### 均线配色原则
- 每条均线使用独特且易区分的颜色
- 颜色饱和度适中，不刺眼
- 在白色和浅色背景上清晰可见

### 布局设计
- **卡片式设计** - 信息分组清晰
- **留白充足** - 视觉不拥挤
- **圆角边框** - 柔和友好
- **阴影效果** - 层次分明

## 📊 技术实现

### 数据处理
```javascript
// CSV数据读取和解析
fetch('./btc-price.csv')
  .then(response => response.text())
  .then(csvText => {
    const lines = csvText.split('\n').filter(line => line.trim());
    const processedData = lines.slice(1).map(line => {
      const [date, price] = line.split(',');
      return {
        date: date.trim(),
        price: parseFloat(price.trim())
      };
    }).reverse();
    setData(processedData);
  });
```

### 均线计算
```javascript
// 简单移动平均线算法
const calculateMA = (dayCount) => {
  const result = [];
  for (let i = 0; i < data.length; i++) {
    if (i < dayCount - 1) {
      result.push('-');
      continue;
    }
    let sum = 0;
    for (let j = 0; j < dayCount; j++) {
      sum += data[i - j].price;
    }
    result.push((sum / dayCount).toFixed(2));
  }
  return result;
};
```

### 性能优化
- **按需计算** - 只计算显示的均线
- **Canvas渲染** - 使用Canvas而非SVG，性能更好
- **平滑曲线** - 减少渲染点数
- **状态管理** - 合理使用React状态，避免不必要的重渲染

## 🔍 使用技巧

### 趋势分析
1. **多头市场识别**
   - 短期均线在长期均线上方
   - 均线呈现多头排列（MA50 > MA200 > MA350 > ...）
   - 价格在主要均线上方

2. **空头市场识别**
   - 短期均线在长期均线下方
   - 均线呈现空头排列
   - 价格在主要均线下方

3. **关键位置**
   - MA200被视为牛熊分界线
   - MA700是长期支撑/压力位
   - MA1400对应比特币4年减半周期

### 信号识别
- **金叉** - MA50上穿MA200，可能是买入信号
- **死叉** - MA50下穿MA200，可能是卖出信号
- **均线密集** - 多条均线靠近，可能预示突破
- **均线发散** - 趋势确立，可能延续
